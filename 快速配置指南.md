### 问题2: 数据库连接错误
**错误信息**: `Access denied for user 'root'@'localhost' (using password: YES)`

**原因**: `.env` 文件中的数据库密码是占位符

**解决方法**:

1. **编辑 .env 文件**
```bash
# 打开 .env 文件
notepad .env
```

2. **修改数据库密码**
```env
# 将这一行：
MYSQL_PASSWORD=your_mysql_password

# 改为您的真实MySQL密码，例如：
MYSQL_PASSWORD=123456
```

3. **如果没有安装MySQL**

如果您还没有安装MySQL，可以选择：

**选项A: 安装MySQL**
- 下载: https://dev.mysql.com/downloads/mysql/
- 安装后创建数据库: `CREATE DATABASE testbrain;`

**选项B: 使用SQLite（临时方案）**
修改 `config/settings.py`:
```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.sqlite3',
        'NAME': BASE_DIR / 'db.sqlite3',
    }
}
```

## 启动步骤

### 1. 配置数据库
```bash
# 编辑 .env 文件，配置真实的数据库密码
notepad .env
```

### 2. 创建数据库（如果使用MySQL）
```sql
CREATE DATABASE testbrain CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 3. 执行数据库迁移
```bash
python manage.py makemigrations
python manage.py migrate
```

### 4. 启动服务
```bash
python main.py
```

## 完整的 .env 配置示例
